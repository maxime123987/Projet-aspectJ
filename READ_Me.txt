projet aspectJ pour le cours d'architecture
Le dossier nommé master contient l'implémentation classique du projet, pour la run il suffit de lancer la classe "SimulationBancaire" dans le dossier src/client. Le resultat sera afficher dans le terminal.

Pour lancer l'implémentation avec aspects,il faut aller dans le dossier aspectJ et lancer le même fichier "SimulationBancaire" dans le dossier src/client. Vous aurez en plus des LOG des opérations effectuées dans le dossier LOGS.
